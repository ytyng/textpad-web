テキスト入力エリアがあるだけのシンプルな Web ページのアプリです。

スマホで使うことをイメージしており、下部のソフトウェアキーボードの領域を除いた上部いっぱいにテキストエリアが表示されます。

さらにその上にはツールバーが表示されます。

主に、テキストエリアを使うことで漢字変換を確認するためのツールです。

PWA として使え、iPhone のホームスクリーンに追加して簡易的なアプリとして使えます。

# 機能

内容は、ローカルストレージに保存することができます。
ローカルストレージに保存する際、テキスト内容の先頭1行目の、20文字目までがタイトルになります。

テキストの内容は自動保存されます。

# メニューバーの内容

- 新規作成
- ファイルリスト
- QRコード読み込み
- QRコード作成

# 仕様
tailwind css でデザインする。

toast は、svelte-sonner を使う。 chat-attendant を参考にする。

QRコードの読み取り、作成は、 qr-code アプリを参考にする。(ただし同じ方式で無くとも良い)

アイコンフォントは、bootstrap icons を使う。これも chat-attendant を参考にする。
